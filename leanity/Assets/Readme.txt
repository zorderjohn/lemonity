Readme

Introduction

Lemonity is a Unity Editor plugin that allows you to navigate around the scene using your hands and Leap Motion.

Requirements
•	Leap Motion device in desktop mode
•	One or more hands
•	Tested on Windows 10 x64

Installation
•	Lemonity requires the Leap Motion Orion service to be installed. You can download it for free here: https://developer.leapmotion.com/get-started
•	Import Lemonity package to your Unity project. 
•	Lemonity includes LeapMotion Unity Core Assets 4.4.0. If you already have this package added to your project, you should avoid importing it by unselecting “LeapMotion” folder. Alternatively, you can remove the folder from the project (it will be in Assets/Lemonity/LeapMotion).

First Steps
1.	Open Lemonity options tab using Unity menu “Window/Lemonity Options”
2.	Choose a working mode in the Lemonity Options tab 
3.	Optionally open Lemonity Debug tab “Window/Lemonity Debug” to see more information.
4.	You are ready to start working with Lemonity.
